# 💍 Свадебное приглашение

Адаптивный одностраничный сайт-визитка (свадебный лендинг) с интерактивной картой, таймером обратного отсчета и интеграцией с Google Таблицами для сбора ответов гостей (RSVP).

## ✨ Особенности проекта
*   **Стильный дизайн**: Минималистичная верстка с тщательно подобранными шрифтами (`Cormorant Garamond` и `Montserrat`).
*   **Таймер**: Динамический счетчик времени, отсчитывающий дни, часы и минуты до торжества.
*   **Интерактивность**: Встроенная карта Яндекс для удобства гостей.
*   **Умная RSVP-форма**: Гости могут подтвердить присутствие, выбрать алкогольные напитки и указать свои пожелания.
*   **Google Sheets Integration**: Все данные из формы мгновенно и без перезагрузки страницы уходят в Google Таблицу.
*   **Полная адаптивность**: Сайт идеально отображается как на экранах компьютеров, так и на смартфонах.

## 🛠️ Технологический стек
*   HTML5 / CSS3 (БЭМ-подобная структура стилей)
*   Чистый JavaScript (Vanilla JS) для работы таймера и AJAX-отправки данных
*   Google Apps Script (в качестве бэкенда для связи с таблицами) [2]

## 📂 Структура файлов
```text
├── index.html   # Разметка страницы, логика таймера и отправки RSVP
├── style.css    # Стили оформления, шрифты и адаптивная сетка
```

## 🚀 Как развернуть проект под себя

### 1. Настройка базы данных (Google Таблицы)
1. Создайте Google Таблицу и добавьте в первую строку заголовки: `guest_name`, `attend`, `alcohol`, `wishes`.
2. Откройте **Расширения -> Apps Script**.
3. Вставьте код обработчика `doPost(e)` (код скрипта находится в истории переписки).
4. Нажмите **Деплой -> Новое развертывание**. Выберите тип **Веб-приложение**, в поле *Кто имеет доступ* обязательно выберите **Все (Anyone)**.
5. Скопируйте полученную ссылку веб-приложения и вставьте её в `index.html` в переменную `WEB_APP_URL`.

### 2. Запуск на GitHub Pages
1. Загрузите файлы `index.html` и `style.css` в ваш публичный репозиторий на GitHub.
2. Перейдите в **Settings** -> **Pages**.
3. В разделе *Branch* выберите ветку `main` (или `master`) и нажмите **Save**.
4. Через 1-2 минуты ваш сайт будет доступен по бесплатному адресу `https://<ваш-логин>.github.io/<имя-репозитория>/`.
